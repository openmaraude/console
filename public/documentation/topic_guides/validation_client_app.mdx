#### Validation d'une application client

Avant de fournir les accès à la plateforme de production, l'équipe le.taxi teste votre application afin de vérifier que les fonctionnalités principales sont correctement implémentées.

Il est attendu que le partenaire passe tous les tests de son côté avant de demander une validation.
